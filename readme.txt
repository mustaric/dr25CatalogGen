Code used to develop the DR25 KOI Catalog
Writen by Susan E. Mullally

Mostly these are scripts that create plots or statistics.
The file script-catalogPlotsWithPublicData.py contains the high level code for most of the plots that are in the DR25 catlaog paper.

The data that these plots rely on can be found here:
